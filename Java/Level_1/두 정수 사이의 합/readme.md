기억해둘 것  
    
    등차수열 공식
    n개의 항에 대해서 첫번째 항 a와 마지막항 b가 주어졌을 때, 등차수열의 합 S = n(a + b) / 2
    n개의 항에 대해서 첫번째 항 a와 공차 d가 주어졌을 때, 등차수열의 합 S = n(2 * a + (n - 1) * d) / 2
