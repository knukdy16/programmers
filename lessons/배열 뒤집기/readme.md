풀이방식    

    여러 방식이 있지만 나는 입력된 배열 자체에서 뒤집는 방식으로 선택
    그러면 for문을 반만 진행하면 된다는 장점이 있음
    왜냐하면 배열의 반까지 진행하면서 뒷 부분도 알아서 바뀌어 있기 때문
    그래서 for문을 배열 길이의 반 까지만 진행하고, for문 내부에서는
    앞부분과 뒷부분을 서로 switch해주도록 구현
